# Wallet Test

A mobile-first React + TypeScript wallet application built with Vite.

## Description

This project is a mobile wallet interface that displays card balance, payment status, daily points, and transaction history. The UI is designed mobile-first, matching a modern wallet app design with clean card layouts and transaction lists.

## Installation

```bash
npm install
```

## Usage

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **React Router DOM** - Routing
- **FontAwesome** - Icons

## Project Structure

```
wallet-test/
├── src/
│   ├── components/     # Reusable UI components
│   ├── pages/          # Page components
│   ├── App.tsx         # Main app component
│   └── main.tsx        # Entry point
├── public/             # Static assets
└── index.html          # HTML template
```

## Features

- Mobile-first responsive design
- Card-based UI layout
- Transaction history list
- Balance and payment status display
- Daily points tracking
